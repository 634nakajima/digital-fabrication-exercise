## 1. パーツのコンポーネント化

### ボディとコンポーネントの違い

これまでは **ボディ** を作って形を作ってきましたが、複数のパーツが相互に動く仕組み（歯車同士が噛み合うなど）を設計するには、パーツを **コンポーネント** として作る必要があります。

| | ボディ | コンポーネント |
|---|---|---|
| 用途 | 1つのパーツ内の形状 | 独立した部品として扱う |
| 移動 | 他のボディと連動 | 個別に移動・回転が可能 |
| ジョイント | 使用不可 | 使用可能 |

::: tip コンポーネントのイメージ
コンポーネントはパソコンの **フォルダー** に似ています。親コンポーネントの中に子コンポーネント、さらに孫コンポーネントと階層化でき、パーツが増えても整理しやすくなります。
:::

### スケッチからコンポーネントを作成する

![コンポーネントの作成](/images/lesson09/component-creation.jpg)

1. `1. パーツのコンポーネント化.f3d` を開きます
2. あらかじめ用意されたスケッチをクリックし、**押し出し**（ショートカット: **E**）を実行します
3. ここで重要：操作のドロップダウンを **「新規ボディ」ではなく「新規コンポーネント」** に変更します
4. 天板を **5 mm** で押し出し、コンポーネント名を「天板」に変更します

### 足を作る（コンポーネントのコピー）

天板と同様に足のスケッチから **新規コンポーネント** として押し出します（高さ **50 mm**）。

足を4本作るには **コピー** を使いますが、ここで重要な違いがあります：

![コンポーネントのコピー](/images/lesson09/component-copy.jpg)

| コピー方法 | 操作 | 特徴 |
|---|---|---|
| **移動/コピー** | 右クリック → 移動/コピー → 「コピーを作成」ON | 元と常に **同期** される（片方を修正するともう片方も変わる） |
| **コピーを貼り付け** | コピー → 右クリック → **コピーを貼り付け** | 完全に **独立** したコピー（片方を修正しても他方に影響しない） |

::: warning 同期コピーに注意
「移動/コピー」で作ったコンポーネントは、片方に穴を開けるともう片方にも穴が開きます。同じパーツを複数使う場合は便利ですが、個別に変更したい場合は「コピーを貼り付け」を使ってください。
:::

## 2. ジョイント（剛性）

### ボトムアップ型とトップダウン型の設計

| アプローチ | 説明 | 向いている場面 |
|---|---|---|
| **ボトムアップ型** | 個々のパーツを先に設計し、後から組み立てる | 汎用パーツの再利用、大規模アセンブリ |
| **トップダウン型** | アセンブリ全体の中で各パーツを設計する | パーツ同士の位置関係が密接な設計 |

今回はボトムアップ型で、先にパーツを作り、**ジョイント** で組み立てるアプローチを学びます。

### 剛性ジョイントでテーブルを組み立てる

`2. ジョイント 剛性.f3d` を開きます。天板と足4本が分離した状態です。

![ジョイントのスナップ点指定](/images/lesson09/joint-rigid-snap.jpg)

1. **アセンブリ → ジョイント** を選択します
2. まず **動かしたい方**（足）のスナップ点をクリックします
3. 次に **行き先**（天板の角）のスナップ点をクリックします
4. タイプが **剛性** になっていることを確認し、**OK** で確定します

![テーブルの組み立て完了](/images/lesson09/table-assembly.jpg)

### スナップ点の指定テクニック

- **面をクリック** → その面の中心にスナップ
- **エッジをクリック** → エッジの中点にスナップ
- **角をクリック** → 頂点にスナップ

::: tip Ctrl（Mac: Cmd）キーで面を固定
スナップ点を選ぶ際、マウスを動かすと別の面に切り替わってしまうことがあります。**Ctrl**（Mac: **Cmd**）キーを押しながら面をクリックすると、その面のスナップ点候補が **固定** されて選びやすくなります。
:::

::: tip 「反転」ボタン
パーツが意図と逆向きにくっついてしまった場合は、ジョイント設定画面の **「反転」** ボタンを押してみてください。
:::

### 練習：箱の組み立て

`3. ジョイント 剛性 練習.f3d` を開きます。6枚の板から箱を組み立てましょう。

![箱の組み立て練習](/images/lesson09/box-assembly.jpg)

**組み立ての順序のコツ：**
1. まず **左右の板** を前面・背面に取り付ける
2. 次に **後ろの板** を取り付ける
3. 最後に **上下の板** を取り付ける

::: warning 組み立て順序に注意
いきなり上下の板をつけようとすると、左右の板がまだ付いていないのでうまくいきません。前面に対して左右を先につけてから、上下の板をつけましょう。
:::

前面・背面には四角い穴と丸い穴が開いています。後でここに棒を通すので、**穴の位置が手前と奥で揃うように** 注意して組み立ててください。

## 3. ジョイント（回転・スライダ）

`4. ジョイント 回転・スライダ.f3d` を開きます。

### 回転ジョイント

**回転ジョイント** は、パーツが1つの軸を中心に回転できるようにします。

1. **アセンブリ → ジョイント** を選択します
2. 回転軸となるスナップ点を指定します
3. モーションタイプを **回転** に設定します
4. **OK** で確定します
5. パーツをドラッグすると、指定した軸を中心に回転します

### スライダジョイント

**スライダジョイント** は、パーツが直線上をスライドできるようにします。

1. **アセンブリ → ジョイント** を選択します
2. スライドの方向を決めるスナップ点を指定します
3. モーションタイプを **スライダ** に設定します
4. **OK** で確定します

### モーション制限の設定

![スライダのモーション制限](/images/lesson09/slider-motion-limit.jpg)

ジョイントの可動範囲を制限して、パーツが意図した範囲だけ動くようにできます。

1. ブラウザパネルやタイムラインでジョイントのアイコンを **ダブルクリック**（または右クリック → **ジョイントの編集**）
2. **モーション** タブで以下を設定します：
   - **最小値**：可動範囲の下限（例: **-23 mm**）
   - **最大値**：可動範囲の上限（例: **-4 mm**）
3. **制限をプレビュー** で確認しながら値を微調整します
4. **OK** で確定します

::: tip 値の微調整
最初からぴったりの値を出すのは難しいです。「制限をプレビュー」を使いながら値を少しずつ変えて、ストッパーの位置とぴったり合うようにしましょう。
:::

### パーツの固定（ピン機能）

ジョイントの操作中に、動いて欲しくないパーツが動いてしまうことがあります。

![ピン止め機能](/images/lesson09/pin-fix.jpg)

- コンポーネントを右クリック → **ピン** を選択すると、そのパーツが完全に固定されます
- ピンで固定したパーツと剛性ジョイントで繋がっているパーツも連動して固定されます
- 解除するには再度右クリック → **ピン解除**

### 練習：箱・回転棒・スライド棒・ストッパーの組み立て

![回転・スライダの練習目標](/images/lesson09/rotation-slider-goal.jpg)

`5. ジョイント 回転・スライダ 練習.f3d` を開きます。以下を組み立てましょう：

1. **箱** を剛性ジョイントで組み立てます（復習）
2. **四角い棒** にストッパー（T字パーツ）を両端に剛性ジョイントで取り付けます
3. 四角い棒を **スライダジョイント** で箱に取り付け、モーション制限を設定します
4. **丸い棒** にストッパーを剛性ジョイントで取り付けます
5. 丸い棒を **回転ジョイント** で箱に取り付けます
6. 完成したら、各パーツを動かして正しく機能するか確認します

### コンポーネントの色分け表示

パーツが増えて区別がつきにくくなったら：

**ユーティリティ → コンポーネントのカラーを表示** をクリックすると、コンポーネントごとに自動で色分けされます。同じコンポーネントのコピーは同じ色になるので、構造が視覚的に分かりやすくなります。

## 4. モーションスタディ

完成した組み立てモデルの動きを **自動でアニメーション再生** する機能です。

![モーションスタディ](/images/lesson09/motion-study.jpg)

1. **アセンブリ → モーションスタディ** を選択します
2. アニメーション表示するジョイントをクリックします
3. タイムライン上に **点を追加** し、角度や位置の値を入力します
   - 例：回転ジョイントに **0° → 360°** を設定
4. **再生** ボタンを押すと動きがアニメーションされます
5. **ループ再生** で繰り返し確認できます

::: tip 速度調整
再生が速すぎる場合は、速度調整スライダーで遅くできます。
:::
