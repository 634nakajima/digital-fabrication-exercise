## 1. パーツのコンポーネント化

### ボディとコンポーネントの違い

Fusion 360 では、3D 形状は **ボディ** として作られますが、複数のパーツを組み合わせるには **コンポーネント** に変換する必要があります。

| | ボディ | コンポーネント |
|---|---|---|
| 用途 | 1つのパーツ内の形状 | 独立した部品として扱う |
| 移動 | 他のボディと連動 | 個別に移動・回転が可能 |
| ジョイント | 使用不可 | 使用可能 |

### .f3d ファイルのアップロード

1. クラスルームで配布された `.f3d` ファイルをダウンロードします
2. Fusion の **データパネル → アップロード** でファイルを読み込みます
3. アップロードしたファイルをダブルクリックして開きます

### スケッチからコンポーネントを作成する

1. スケッチでパーツの断面を描きます
2. 押し出す前に、**ブラウザ** パネルでスケッチを右クリック → **新しいコンポーネントとして押し出し** を選択します
3. これにより、押し出したボディが自動的にコンポーネントとして作成されます

### 既存のボディをコンポーネント化する

1. **ブラウザ** パネルで変換したいボディを右クリックします
2. **コンポーネントの作成** を選択します
3. ボディがコンポーネントとしてラップされ、ジョイントが使えるようになります

### コンポーネントの2種類のコピー

| コピー方法 | 特徴 |
|---|---|
| **コピー＆ペースト** | 独立したコピー。片方を変更してももう片方に影響しない |
| **リンクコピー** | 元のコンポーネントとリンク。片方を変更するともう片方にも反映される |

::: tip リンクコピーの使いどころ
同じ形状のパーツを複数使う場合（ネジ、歯車の歯など）はリンクコピーが便利です。1つを修正するとすべてに反映されます。
:::

## 2. ジョイント（剛性）

### ボトムアップ型とトップダウン型の設計

| アプローチ | 説明 | 向いている場面 |
|---|---|---|
| **ボトムアップ型** | 個々のパーツを先に設計し、後から組み立てる | 汎用パーツの再利用、大規模アセンブリ |
| **トップダウン型** | アセンブリ全体の中で各パーツを設計する | パーツ同士の位置関係が密接な設計 |

### 剛性ジョイントの基本

**剛性ジョイント** は、2つのコンポーネントを固定して動かないようにする接合です。

1. **アセンブリ → ジョイント** を選択します
2. **コンポーネント1** のスナップ点（面の中心、エッジの中点など）をクリックします
3. **コンポーネント2** のスナップ点をクリックします
4. モーションタイプが **剛性** になっていることを確認します
5. **OK** で確定すると、2つのパーツがぴったり接合されます

### スナップ点の指定

ジョイントを正確に配置するために、以下のスナップ点を活用します：

- **面の中心**：面をクリックすると自動で中心が選ばれる
- **エッジの中点**：エッジをクリックすると中点がスナップされる
- **頂点**：角をクリックすると頂点にスナップされる

::: tip Ctrl（Mac: Cmd）キーで面を固定
ジョイント設定中に **Ctrl**（Mac: **Cmd**）キーを押しながら面をクリックすると、その面を基準面として固定できます。パーツの向きを揃えるのに便利です。
:::

### 練習：箱の組み立て

配布された `.f3d` ファイルを使って、板状のパーツから箱を組み立てます。

1. 各パーツをコンポーネント化します
2. **底板** を固定し、基準とします
3. **剛性ジョイント** を使って、側面の板を1枚ずつ底板に接合します
4. すべての側面を接合して箱を完成させます

## 3. ジョイント（回転・スライダ）

### 回転ジョイント

**回転ジョイント** は、パーツが1つの軸を中心に回転できるようにします。

1. **アセンブリ → ジョイント** を選択します
2. 回転軸となるスナップ点を指定します
3. モーションタイプを **回転** に設定します
4. **OK** で確定します
5. パーツをドラッグすると、指定した軸を中心に回転します

### スライダジョイント

**スライダジョイント** は、パーツが直線上をスライドできるようにします。

1. **アセンブリ → ジョイント** を選択します
2. スライドの方向を決めるスナップ点を指定します
3. モーションタイプを **スライダ** に設定します
4. **OK** で確定します

### モーション制限の設定

ジョイントの可動範囲を制限して、パーツが意図した範囲だけ動くようにできます。

1. ブラウザパネルで設定したいジョイントを右クリック → **ジョイントの編集** を選択します
2. **モーション** タブで以下を設定します：
   - **最小値**：可動範囲の下限
   - **最大値**：可動範囲の上限
3. **OK** で確定します

### 練習：箱・回転棒・スライド棒・ストッパーの組み立て

配布された `.f3d` ファイルを使って、以下の組み立てを行います：

1. **箱** を剛性ジョイントで組み立てます（復習）
2. 箱に **回転棒** を回転ジョイントで取り付けます
3. **スライド棒** をスライダジョイントで取り付けます
4. スライド棒に **モーション制限** を設定し、ストッパーの範囲内で動くようにします
5. 完成したら、各パーツを動かして正しく機能するか確認します
