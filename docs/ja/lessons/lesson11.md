## 1. カム機構とは

**カム機構** は、回転運動を往復運動（上下動など）に変換する機構です。からくり人形やオルゴールなどに広く使われています。

### カム機構の構成

| 部品 | 役割 |
|------|------|
| **カム** | 回転する部品。形状によって運動パターンが変わる |
| **フォロワ** | カムの表面に沿って動く部品。上下運動などの出力を行う |
| **フレーム** | カムとフォロワを支える構造体 |

### カムの種類

| カムの形状 | 動きの特徴 |
|---|---|
| **円板カム（偏心カム）** | なめらかな上下動。最もシンプル |
| **ハート型カム** | 等速で上昇・等速で下降。一定リズムの動き |
| **卵型カム** | ゆっくり上昇・すばやく下降（またはその逆） |
| **不定形カム** | 自由な動きを設計可能。創作向き |

## 2. Fusion でカム機構を設計する

### カムの作成

1. **XY 平面** にスケッチを作成します
2. カムの形状を描きます：
   - シンプルな偏心カムの場合：中心からオフセットした位置に **円** を描きます
   - より複雑な形状：**スプライン** や **円弧** を組み合わせて描きます
3. 回転軸の穴を描きます（例：直径 **5 mm** の円）
4. スケッチを **押し出し** て、カムのボディにします（例：厚さ **5 mm**）
5. コンポーネント化し、**回転ジョイント** を設定します

### フォロワの作成

1. カムの上方にスケッチを作成します
2. フォロワの形状を描きます（棒状が基本）
3. 押し出してボディにし、コンポーネント化します
4. **スライダジョイント** を設定し、上下にのみ動けるようにします

### カムとフォロワの接触

1. カムを回転させたとき、フォロワがカムの表面に沿って上下に動くことを確認します
2. **接触セット** を使って、カムとフォロワの物理的な接触を設定できます：
   - **ソリッド → 修正 → 接触セット** から設定します
3. カムを回転させて、フォロワが正しく追従するか確認します

::: tip フォロワが追従しない場合
接触セットが正しく設定されているか確認してください。また、フォロワのスライダジョイントの方向がカムの動きと合っているかも確認しましょう。
:::

## 3. 揺動クランク機構

**揺動クランク機構** は、回転運動を揺動（往復回転）運動に変換する機構です。カム機構とは異なるアプローチで往復運動を実現します。

### 構成部品

| 部品 | 役割 |
|------|------|
| **クランク** | 一定方向に回転する入力部品（ハンドルで回す） |
| **コネクティングロッド** | クランクとロッカーをつなぐリンク |
| **ロッカー** | 一定の角度範囲で揺動する出力部品 |
| **フレーム** | クランクとロッカーの回転軸を固定する |

### Fusion で揺動クランクを設計する

1. **フレーム** を作成します
   - クランクの回転軸とロッカーの回転軸の2つの穴を持つ板
   - 2つの軸間距離が重要です

2. **クランク** を作成します
   - 回転軸の穴と、コネクティングロッドとの接続点を持つアーム
   - クランクの長さ（回転半径）が揺動の幅を決めます

3. **コネクティングロッド** を作成します
   - 両端に回転軸の穴がある棒状の部品
   - 長さはクランクとロッカーの位置関係に合わせて決めます

4. **ロッカー** を作成します
   - フレームに固定された軸を中心に揺動するアーム
   - 先端に人形の腕や首など、動かしたい部品を取り付けます

5. 各部品を **回転ジョイント** で接続します：
   - クランク ↔ フレーム：回転ジョイント
   - クランク ↔ コネクティングロッド：回転ジョイント
   - コネクティングロッド ↔ ロッカー：回転ジョイント
   - ロッカー ↔ フレーム：回転ジョイント

6. クランクを回して、ロッカーが揺動するか確認します

::: warning リンク比の注意
クランクの長さ、コネクティングロッドの長さ、2つの軸間距離のバランスが重要です。バランスが悪いと機構がロックして動かなくなります。配布ファイルの寸法を参考にしてください。
:::

## 4. カム機構と揺動クランクの比較

| 特徴 | カム機構 | 揺動クランク機構 |
|------|----------|-----------------|
| **動きの自由度** | カム形状で自由に設計可能 | リンク比で揺動角度を調整 |
| **部品点数** | 少ない（カム＋フォロワ） | 多い（クランク＋ロッド＋ロッカー） |
| **滑らかさ** | カム形状に依存 | リンク機構のため比較的なめらか |
| **製作難易度** | カム形状の加工が必要 | 穴あけとピン接合が中心 |
| **からくりでの用途** | 人形の上下動、首振り | 腕振り、足踏み、太鼓叩き |

## 練習課題

配布された `.f3d` ファイルを使って以下の練習を行いましょう：

1. **`12. カム機構.f3d`** を開き、カムとフォロワの関係を確認する
2. カムの形状を変更して、フォロワの動きがどう変わるか観察する
3. **`13. 揺動クランク.f3d`** を開き、各部品の接続を確認する
4. **`13. 揺動クランク（完成版）.f3d`** と比較して、自分で組み立ててみる
5. どちらの機構が自分のからくり作品に適しているか検討する
