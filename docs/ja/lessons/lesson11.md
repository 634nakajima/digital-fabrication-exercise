## 機構の種類について

前回の歯車機構と今回のカム機構の他にも、からくり作品で活用できる機構はたくさんあります。

| 機構 | 特徴 |
|------|------|
| **スパーギア** | 平行な軸間で回転を伝達（前回学習済み） |
| **かさ歯車（ベベルギア）** | 直角に交わる軸間で回転を伝達 |
| **ウォームギア** | 高い減速比が得られる。逆転防止機能がある |
| **ラック＆ピニオン** | 回転運動を直線運動に変換 |
| **カム機構** | 回転を上下動に変換（今回学習） |
| **スライダクランク** | 回転を直線往復運動に変換（ピストン運動） |
| **揺動クランク** | 回転を揺動運動に変換（今回学習・参考） |

## 1. カム機構とは

**カム機構** は、回転運動を往復運動（上下動など）に変換する機構です。からくり人形やオルゴールなどに広く使われています。

### カム機構の構成

| 部品 | 役割 |
|------|------|
| **カム** | 回転する部品。形状によって運動パターンが変わる |
| **フォロワ（従動節）** | カムの表面に沿って動く部品。上下運動の出力を行う |
| **フレーム** | カムとフォロワを支える構造体 |

### カムの種類

| カムの形状 | 動きの特徴 |
|---|---|
| **円板カム（偏心カム）** | なめらかな上下動。最もシンプル |
| **ハート型カム** | 等速で上昇・等速で下降。一定リズムの動き |
| **卵型カム** | ゆっくり上昇・すばやく下降（またはその逆） |
| **不定形カム** | 自由な動きを設計可能。創作向き |

## 2. Fusion でカム機構を設計する

`12. カム機構.f3d` を開きます。フレーム、軸、ハンドルがあらかじめ用意されています。

### カムの作成

![カムのスケッチ](/images/lesson11/cam-sketch.jpg)

1. 適当な平面にスケッチを作成します
2. **円** を描きます（例：直径 **22.5 mm**）
3. 円の中心に **軸穴** を描きます（直径 **2 mm**）
4. 重要：**軸穴の位置を中心からずらします**

::: tip 偏心量と上下幅の関係
軸穴を円の中心から **どれだけずらすか** で、上下の動きの幅が決まります。

- ずらした距離の **2倍** が上下動の幅になります
- 例：中心から **6.25 mm** ずらすと → 上下 **12.5 mm** の往復運動

ずらす距離が大きいほど動きは大きくなりますが、機構への負荷も大きくなります。
:::

5. **スケッチ寸法** で穴の位置を正確に設定します（例：中心から **6.25 mm**）
6. スケッチを **押し出し** ます（厚さ **2.5 mm**、MDF想定）
7. **新規コンポーネント** として作成し、名前を「カム」にします

### カム受け面の作成

カムからの力を受け止める面は、なるべく広くする必要があります。

1. 新しいスケッチで **正方形** を描きます（例：**15 × 15 mm**、中心に **2 mm** の穴）
2. **2.5 mm** で押し出し、**新規コンポーネント** にします（名前：「カム面」）
3. **剛性ジョイント** で軸棒の先端に取り付けます
4. ジョイント後にずれが生じた場合は、右クリック → **ジョイントの編集** でオフセットを微調整します（例：**-0.75 mm**）

::: warning 摩擦と軸曲がりへの対策
カム機構ではカムとフォロワの間に摩擦が発生します。実際に制作する際は：
- **潤滑スプレー** を塗布する
- **アクリル板** など滑りの良い素材を使う
- カムの **幅を増やす**（MDF を重ねる）
- カム受け面を **大きくする**（接触面積を増やす）
- 軸の **非支持区間を短くする**（スペーサーで隙間を埋める）。軸が曲がるとカムが正しく接触しなくなります
:::

### 軸（フォロワ）のジョイント設定

![カム機構の組み立て](/images/lesson11/cam-mechanism.jpg)

フォロワの棒は上下にのみ動けるようにします：

1. 軸と天板を **スライダジョイント** で接続します
2. 方向を **上下方向（Y軸）** に設定します
3. この時点ではまだカムと連動しません

### 接線関係でカムの動きをシミュレーション

![接線関係の設定](/images/lesson11/tangent-relation.jpg)

カムの曲面とフォロワの面を常に接触させるには、**モーションリンクではなく「接線関係」** を使います。

::: tip なぜモーションリンクではダメなのか
ギアの場合は回転角度の比で連動できますが、カム機構では「何度回転したら何mm動く」が一定ではありません。カムの形状によって変化するため、**接線関係** を使って「常に2つの面が接している」という制約を設定します。
:::

1. **アセンブリ → 接線関係** を選択します
2. **カムの外周面** をクリックします
3. **フォロワのカム受け面** をクリックします
4. **OK** で確定すると、自動的に正しい位置関係が計算されます

設定後、ハンドルの回転ジョイントをドラッグして回すと、カム受け面が常にカムに接した状態で上下に動きます。

## 3. 揺動クランク機構（参考）

::: tip 参考資料
揺動クランク機構は授業の進捗に応じて扱います。時間の都合で省略する場合もありますが、最終課題で活用することもできます。完成見本ファイルがクラスルームにあるので、興味のある人は自習してみてください。
:::

**揺動クランク機構** は、回転運動を揺動（往復回転）運動に変換する機構です。カム機構とは異なるアプローチで往復運動を実現します。

### 構成部品

![揺動クランクの部品](/images/lesson11/crank-parts.jpg)

| 部品 | 役割 |
|------|------|
| **クランク** | 一定方向に回転する入力部品（ハンドルで回す） |
| **コネクティングロッド** | クランクとロッカーをつなぐリンク |
| **ロッカー** | 一定の角度範囲で揺動する出力部品 |
| **フレーム** | クランクとロッカーの回転軸を固定する |

### Fusion で揺動クランクを組み立てる

![揺動クランクの組み立て](/images/lesson11/crank-assembly.jpg)

`13. 揺動クランク.f3d` を開きます。各パーツは作成済みですが、ジョイントが設定されていません。

1. **フレーム** を固定します（ピン止め推奨）
2. **クランク** をフレームに **回転ジョイント** で接続します
3. **コネクティングロッド** の一端をクランクに **回転ジョイント** で接続します
4. **コネクティングロッド** の他端をロッカーに **回転ジョイント** で接続します
5. **ロッカー** をフレームに **回転ジョイント** で接続します

すべてのジョイントが **回転ジョイント** であることがポイントです。

![揺動クランクのシミュレーション](/images/lesson11/crank-simulation.jpg)

組み立てが完了したら、クランクをドラッグして回してみましょう。ロッカーが一定の角度範囲で揺動することを確認できます。

::: warning リンク比の注意
クランクの長さ、コネクティングロッドの長さ、2つの軸間距離のバランスが重要です。バランスが悪いと機構がロックして動かなくなります。**`13. 揺動クランク（完成版）.f3d`** と比較して確認してください。
:::

## 4. カム機構と揺動クランクの比較

| 特徴 | カム機構 | 揺動クランク機構 |
|------|----------|-----------------|
| **出力の動き** | 直線往復（上下動） | 揺動（往復回転） |
| **動きの自由度** | カム形状で自由に設計可能 | リンク比で揺動角度を調整 |
| **部品点数** | 少ない（カム＋フォロワ） | 多い（クランク＋ロッド＋ロッカー） |
| **滑らかさ** | カム形状に依存 | リンク機構のため比較的なめらか |
| **製作時の注意** | 摩擦対策が必要 | 穴あけとピン接合の精度が重要 |
| **からくりでの用途** | 人形の上下動、首振り | 腕振り、足踏み、太鼓叩き |
