# 第13回 作品制作 2：試作品の組み立て

## 今回のゴール

- 出力したパーツを組み立てて試作品を作る
- 機構が正しく動作するか検証する
- 問題点を特定し、改善計画を立てる

---

## 今回の作業

### 1. パーツの確認と仕上げ

1. 前回出力したパーツを確認します
2. 必要に応じて以下の後処理を行います：
   - **バリ取り**：カッターやヤスリで余分な部分を除去
   - **サポート材の除去**：3Dプリント品の場合
   - **穴の調整**：軸穴がきつい場合はドリルやヤスリで微調整
   - **表面の研磨**：ヤスリで表面をなめらかに

### 2. 試作品の組み立て

1. 設計した順序でパーツを組み立てます
2. 以下の点を確認しながら進めます：
   - パーツ同士がきちんとはまるか
   - 回転軸はスムーズに回るか
   - 歯車やカムが正しくかみ合うか
   - フレーム（土台）は十分な強度があるか

::: tip 組み立てがうまくいかない場合
- 穴がきつい → ドリルやヤスリで広げる
- 穴がゆるい → 接着剤で固定するか、パーツを再設計・再出力
- 軸がぶれる → 軸受け部分の精度を見直す
- 歯車がかみ合わない → 中心間距離やバックラッシュを確認
:::

### 3. 動作検証

1. ハンドルを回すなどして機構を動かします
2. 以下を確認します：
   - 意図した動きになっているか
   - スムーズに動くか（引っかかりがないか）
   - 耐久性に問題はないか

### 4. 改善計画

動作検証で見つかった問題点を整理し、次回までの改善計画を立てます。
