## 1. 歯車の基本知識

歯車は回転運動を伝達する機構部品です。からくり作品では、回転の速度や方向を変えるために使います。

| 用語 | 説明 |
|------|------|
| **モジュール** | 歯の大きさを表す値。かみ合う歯車同士は **同じモジュール** にする |
| **歯数** | 歯車の歯の数。歯数が多いほど直径が大きくなる |
| **ピッチ円直径** | **モジュール × 歯数** で求まる。かみ合いの基準となる円の直径 |
| **圧力角** | 一般的に 20° が標準 |

::: tip ピッチ円直径の計算
例：モジュール 2、歯数 18 の場合 → ピッチ円直径 = 2 × 18 = **36 mm**

2つの歯車の **中心間距離** は `(ピッチ円直径1 + ピッチ円直径2) ÷ 2` で求まります。
:::

## 2. スパーギアを作成する

![スパーギアのパラメータ設定](/images/lesson10/spur-gear-params.jpg)

`6. ギアを作る.f3d` を開きます。軸（直径 2 mm、高さ 5 mm）と土台があらかじめ用意されています。

### スパーギアの作り方

1. **ユーティリティ → アドイン → スパーギア** を選択します
2. 以下のパラメータを設定します：

| パラメータ | 値（例） | 説明 |
|-----------|---------|------|
| **モジュール** | 2 | 歯の大きさ |
| **ピッチ円直径** | 36 mm | モジュール × 歯数 で決まる |
| **歯数** | 18（= 36 ÷ 2） | ピッチ円直径 ÷ モジュール |
| **厚さ** | 2.5 mm | MDF の板厚に合わせる |
| **穴直径** | 2 mm | 軸の太さに合わせる |

3. **OK** でギアが生成されます

::: tip スパーギアが見つからない場合
Fusion のバージョンによって場所が異なります。画面上部の **検索バー**（虫眼鏡アイコン）に「スパーギア」と入力すると見つかります。
:::

### 歯数の求め方

ピッチ円直径が指定されている場合の歯数の計算：

```
歯数 = ピッチ円直径 ÷ モジュール
```

例：ピッチ円直径 24 mm、モジュール 2 → 歯数 = 24 ÷ 2 = **12**

### 2つの歯車を配置する

![歯車のかみ合わせ](/images/lesson10/gear-mesh.jpg)

1. 2つのギアを作成します（例：歯数 18 と歯数 12）
2. それぞれを **回転ジョイント** で軸に取り付けます：
   - ギアの穴の中心 → 軸の先端をスナップ点に指定
   - モーションタイプを **回転** に設定
3. 軸と土台は **剛性ジョイント** で固定しておきます

::: warning 軸間距離に注意
テンプレートの土台には穴が2つ開いていますが、この穴の距離がピッチ円直径の合計の半分に合っている必要があります。合っていないとギアが噛み合いません。
:::

## 3. モーションリンク

![モーションリンクの設定](/images/lesson10/motion-link.jpg)

2つの歯車を連動させて、一方を回すともう一方も回転するようにします。

1. **アセンブリ → モーションリンク** を選択します
2. 連携させたい **回転ジョイント** を2つクリックします
3. **反転** ボタンを押して、逆方向に回転するようにします

### 回転比の計算

歯車が1回転する間に何度回るかは、**歯数の比** で決まります。

例：歯数 18 のギアと歯数 12 のギアの場合

- ギア1（歯数 18）が **360°** 回転すると
- ギア2（歯数 12）には歯が **12個** しかないので、12/18 = 2/3 の回転
- 360° × 2/3 = **240°**

つまり、モーションリンクの設定で：
- ジョイント1：**360°**
- ジョイント2：**240°**（反転ON）

![3つのギアの連動](/images/lesson10/three-gears.jpg)

::: tip 歯の初期位置の調整
ギア同士がうまく噛み合わない場合は、回転ジョイントを右クリック → **ジョイントの編集** で **初期位置** の角度を数度ずらして微調整してください。
:::

## 4. ハンドル（クランクアーム）を作る

![ハンドルの作成](/images/lesson10/handle-creation.jpg)

歯車を手で回すためのハンドルを作成します。

1. 歯車の側面にスケッチを作成します
2. **線分** や **円弧** を使ってアーム形状を描きます
3. アームの先端に **円** を描き、握りの部分にします
4. スケッチを **押し出し** て（厚さ 2.5 mm）ボディを作成します
5. **新規コンポーネント** として作成し、ジョイントで歯車に接合します

## 5. 寸法・位置の調整

![寸法の調整](/images/lesson10/dimension-adjust.jpg)

パーツを正確に配置するためのテクニックです。

### フィーチャー履歴からの修正

1. 画面下部の **タイムライン** でスケッチや押し出しのフィーチャーを **ダブルクリック** します
2. 寸法を変更し、**OK** で確定すると以降の操作に自動で反映されます

### ジョイントの編集

1. ジョイントのアイコンを **右クリック** → **ジョイントの編集**
2. **位置合わせ** タブで角度やオフセットを微調整
3. **モーション** タブで最大値・最小値を設定

![位置の調整](/images/lesson10/position-adjust.jpg)

### コンポーネントの階層化

パーツが増えてきたら、関連するコンポーネントを親コンポーネントにまとめましょう：

1. **右クリック → 新規コンポーネント** で親コンポーネントを作成（例：「ギア1」「ギア2」「ギア3」）
2. 各パーツ（歯車、軸、土台）をドラッグして親コンポーネントに入れる
3. これにより階層構造が整理され、見やすくなります

## 6. DXF ファイルの書き出し

レーザーカッターで材料を切り出すために、2D の DXF ファイルを書き出します。

### 書き出し手順

1. 書き出したいパーツの **平面（切り出す面）** を選択します
2. **スケッチ → スケッチを作成** でその面にスケッチを開始します
3. **スケッチ → 作成 → プロジェクト** で、ボディのエッジをスケッチ平面に投影します
4. **スケッチを終了** します
5. スケッチを右クリック → **DXF として保存** を選択します

::: warning レーザーカッター用の注意点
- 切り出し時の **切りしろ（kerf）** を考慮し、必要に応じてオフセットを加えましょう
- 材料の厚みに合わせて、はめ合い部分の寸法を微調整してください
- MDF 2.5 mm 板を想定して設計するのが基本です
:::
