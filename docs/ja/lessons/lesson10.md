## 1. 歯車を作る

### 歯車の基本知識

歯車は回転運動を伝達する機構部品です。からくり作品では、回転の速度や方向を変えるために使います。

| 用語 | 説明 |
|------|------|
| **モジュール** | 歯の大きさを表す値。かみ合う歯車同士は同じモジュールにする |
| **歯数** | 歯車の歯の数。歯数が多いほど直径が大きくなる |
| **ピッチ円直径** | モジュール × 歯数 で求まる。かみ合いの基準となる円の直径 |
| **圧力角** | 一般的に 20° が標準 |

### Fusion でスパーギア（平歯車）を作成する

Fusion 360 にはギアを自動生成する機能があります。

1. **ソリッド → 作成 → スパーギア** を選択します（または **ユーティリティ → アドイン → スパーギア** から）
2. 以下のパラメータを設定します：
   - **モジュール**：任意（例：2）
   - **歯数**：任意（例：20）
   - **厚さ**：任意（例：5 mm）
   - **圧力角**：20°
   - **バックラッシュ**：レーザーカッターで切り出す場合は材料の厚みに応じて調整
3. **OK** で歯車が生成されます

::: tip スパーギアが見つからない場合
Fusion のバージョンによって場所が異なります。**検索バー**（画面上部の虫眼鏡アイコン）に「スパーギア」と入力すると見つかります。
:::

### 2つの歯車をかみ合わせる

1. 同じモジュールで歯数の異なる歯車を2つ作成します
2. 2つの歯車の中心間距離は **（歯数1 + 歯数2）÷ 2 × モジュール** で求めます
3. **ジョイント** の **回転** で各歯車の中心軸を回転軸に設定します
4. **モーションリンク** を使うと、2つの歯車を連動させて回すことができます

## 2. ハンドル（クランクアーム）を作る

歯車を手で回すためのハンドルを作成します。

1. 歯車の側面にスケッチを作成します
2. **線分** や **円弧** を使ってアーム形状を描きます
3. アームの先端に **円** を描き、握りの部分にします
4. スケッチを **押し出し** てアームのボディを作成します
5. 歯車と結合するか、コンポーネント化してジョイントで接合します

::: tip ハンドルの長さ
ハンドルが長いほど小さな力で回せますが、回転半径が大きくなります。作品のサイズに合わせて調整しましょう。
:::

## 3. 歯車を直角に組み合わせる

**かさ歯車（ベベルギア）** を使うと、回転軸を直角に変換できます。

### かさ歯車の特徴

- 2つの軸が直角に交わる場合に使用
- 入力軸と出力軸の回転方向が変わる
- からくり作品では、水平の回転を垂直に変換するときなどに活用

### Fusion でかさ歯車を作成する

1. 2つの平歯車を、軸が直角になるように配置します
2. 歯の形状を調整して、かみ合うようにします
3. **回転ジョイント** でそれぞれの軸を設定します
4. **モーションリンク** で連動させます

## 4. 寸法・位置の調整

パーツを正確に配置するためのテクニックです。

### パラメータを使った寸法管理

1. **修正 → パラメータの変更** を選択します
2. よく使う寸法を **ユーザーパラメータ** として定義します（例：`gear_module = 2`）
3. スケッチ寸法やフィーチャーのサイズにパラメータ名を入力すると、一括で変更できます

### 位置合わせ

- **整列** コマンド：面同士を揃える
- **ジョイントの原点** を使って正確な位置関係を設定
- **計測** ツールで距離や角度を確認しながら調整

## 5. DXF ファイルの書き出し

レーザーカッターで材料を切り出すために、2D の DXF ファイルを書き出します。

### 書き出し手順

1. 書き出したいパーツの **平面（切り出す面）** を選択します
2. **スケッチ → スケッチを作成** でその面にスケッチを開始します
3. **スケッチ → 作成 → プロジェクト** で、ボディのエッジをスケッチ平面に投影します
4. **スケッチを終了** します
5. スケッチを右クリック → **DXF として保存** を選択します

::: warning レーザーカッター用の注意点
- 線の色やレイヤーの設定はレーザーカッターのソフトに合わせて調整が必要な場合があります
- 切り出し時の **切りしろ（kerf）** を考慮し、必要に応じてオフセットを加えましょう
- 材料の厚みに合わせて、はめ合い部分の寸法を微調整してください
:::
