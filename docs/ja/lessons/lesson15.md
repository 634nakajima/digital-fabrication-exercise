## 発表の流れ

### 1. 作品発表

1人ずつ作品を発表します。

**発表内容（1人あたり 3〜5 分程度）：**

1. **作品名**とコンセプトの紹介
2. **実演**：作品を動かして見せる
3. **使用した機構** の説明
4. **工夫したポイント** と **苦労した点**
5. 質疑応答

::: tip 発表のポイント
- 作品を動かしながら説明すると伝わりやすいです
- 「なぜこの機構を選んだか」「何を見立てたか」を話すと独創性が伝わります
- 失敗や試行錯誤のエピソードも共有しましょう。それも学びの一部です
:::

### 2. 講評

- 教員からのフィードバック
- 受講生同士の感想・コメント

### 3. 全体のまとめ

授業全体を振り返り、デジタルファブリケーションの学びを整理します。
