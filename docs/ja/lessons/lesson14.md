# 第14回 作品の完成・動画撮影

## 今回のゴール

- 試作品を改善し、作品を完成させる
- 作品の動画を撮影する
- 作品概要ドキュメントを作成する

---

## 今回の作業

### 1. 試作品の改善

前回の動作検証で見つかった問題点を修正します。

- パーツの再設計・再出力が必要な場合は早めに取りかかりましょう
- 微調整で解決できる場合は、ヤスリやドリルで対応します

### 2. 仕上げ・装飾

作品の完成度を高めるための仕上げ作業を行います。

| 工夫 | 方法 |
|------|------|
| **表面処理** | ヤスリがけ、パテ埋め |
| **塗装** | アクリル絵の具、スプレー塗料 |
| **装飾** | シール、テープ、布、紙など異素材の活用 |
| **ベースの整備** | 土台をきれいに仕上げ、安定性を確保 |

::: tip 評価ポイント「完成度」
表面処理・塗装・装飾・異素材の利用などの工夫は、評価の「完成度」に含まれます。時間の許す限り仕上げにこだわりましょう。
:::

### 3. 動画の撮影

作品が動いている様子を動画で撮影します。

1. **明るい場所** で撮影します
2. 作品全体が映るように **三脚やスマホスタンド** を使うと安定します
3. 以下の内容を含めてください：
   - 作品の全体像
   - ハンドルを回して機構が動く様子
   - 特に工夫したポイントのアップ
4. 動画の長さは **30秒〜1分程度** が目安です

### 4. 作品概要ドキュメントの作成

以下の内容を含むドキュメント（Word や Google ドキュメント等）を作成します。

- **作品名**
- **コンセプト**：どのような発想・見立てで作品を作ったか
- **使用した機構**：歯車、カム、揺動クランクなど
- **工夫したポイント**：設計・造形・仕上げで工夫した点
- **苦労した点と解決方法**
