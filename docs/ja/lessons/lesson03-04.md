## Part 1：Fusionの基本操作

### 1-1. Fusionの起動とマウス操作

1. Fusion を起動し、新しいプロジェクトを作成します
2. **データパネル**（左上）からプロジェクトを管理できることを確認します
3. マウスの基本操作を練習します：
   - **ホイールドラッグ**：視点の平行移動（パン）
   - **ホイール回転**：ズームイン・ズームアウト
   - **Shift + ホイールドラッグ**：視点の回転（オービット）
   - 画面右上の **ViewCube** をクリックすると正面・上面などの定位置に移動できます

::: tip マウス操作のコツ
最初に視点操作に慣れることが大切です。自由に回転・ズームできるようになるまで練習しましょう。
:::

### 1-2. 直方体・円柱の作成

**直方体を作成する：**

1. メニューから **ソリッド → 作成 → 直方体** を選択します
2. 原点をクリックして配置の基準点を決めます
3. 以下のサイズを入力します：
   - 幅：**25 mm**
   - 奥行き：**25 mm**
   - 高さ：**25 mm**
4. **OK** を押して確定します

**円柱を作成する：**

1. **ソリッド → 作成 → 円柱** を選択します
2. 原点をクリックします
3. 以下のサイズを入力します：
   - 直径：**50 mm**
   - 高さ：**15 mm**
4. **OK** で確定します

### 1-3. ボディの結合・切り取り

Fusion では複数のボディを **結合**（合体）したり、**切り取り**（穴を開ける）ことができます。

**結合の練習：**

1. 先ほど作成した円柱の中心上に、新しい円柱を作成します：
   - 直径：**30 mm**、高さ：**10 mm**
2. **ソリッド → 修正 → 結合** を選択します
3. **ターゲットボディ** に元の円柱、**ツールボディ** に新しい円柱を指定します
4. 操作を **結合** に設定し、**OK** で確定します

**切り取りの練習：**

1. 結合した立体の上面中心に、新しい円柱を作成します：
   - 直径：**20 mm**、高さ：ボディ全体を貫通する長さ
2. **結合** コマンドで操作を **切り取り** に設定します
3. **OK** で確定すると、中心に穴が開きます

### 1-4. フィーチャー履歴と編集

Fusion では操作がすべて **タイムライン**（画面下部）に記録されます。過去の操作をいつでも修正できます。

1. 画面下部のタイムラインで、上段の円柱を作成したフィーチャーをダブルクリックします
2. 直径を **40 mm** に変更します
3. **OK** で確定すると、以降の操作もすべて自動で更新されます

::: warning 注意
タイムラインの順序を入れ替えると、依存関係のある操作がエラーになることがあります。変更する際は前後の操作への影響を確認しましょう。
:::

### 1-5. 表示制御・保存・バージョン管理

**表示・非表示の切り替え：**

1. 画面左の **ブラウザ** パネルで、直方体のボディの目のアイコンをクリックして非表示にします
2. 円柱を右クリック → **不透明度** を **50%** に変更します

**保存とバージョン管理：**

1. **Ctrl + S**（Mac: **Cmd + S**）でプロジェクトを保存します
2. 直方体の上に 1辺 **10 mm** の立方体を結合し、再度保存します
3. **ファイル → バージョン履歴** から、保存前のバージョンを開きます
4. そのバージョンを **最新として保存** すると、変更を巻き戻せます

### 1-6. スケッチの作成

スケッチは 2D の図形を描く機能です。スケッチから 3D ボディを作成します。

**線分と曲線：**

1. **スケッチ → 作成 → 線分** を選択します
2. 任意の平面をクリックしてスケッチモードに入ります
3. 線分を使って **平行四辺形** を描きます
4. 続けて、直線と曲線を組み合わせた **閉曲線** を描きます
5. **スケッチ → 作成 → フィット点スプライン** で、自由な曲線を描きます

**トリム・オフセット・パターン：**

| 操作 | 手順 |
|------|------|
| **トリム** | 大小 2 つの円を描き、トリムで不要な線を削除して三日月を作成 |
| **オフセット** | 三日月の内側に小さな三日月をオフセットで作成 |
| **ミラー** | 半分のハートを描き、ミラーで左右対称にしてハートを作成 |
| **円形状パターン** | 直径 10 mm の円を、任意の個数で円周上に配置 |
| **格子状パターン** | 1辺 10 mm の正方形を 3×3 個、幅 5 mm の余白を空けて配置 |
| **スケッチ寸法** | 任意の辺を選択し、寸法を設定・変更 |

### 1-7. 押し出し

スケッチで描いた 2D 図形を 3D に変換します。

1. スケッチの閉じた図形（プロファイル）を選択します
2. **ソリッド → 作成 → 押し出し** を選択します
3. 高さを指定して **OK** で確定します

### 1-8. プロジェクトファイルのエクスポート

作成したモデルを外部ファイルとして書き出します。

**`.f3d` ファイル（Fusion ネイティブ形式）：**

1. **ファイル → エクスポート** を選択します
2. ファイル形式で **Fusion Archive (.f3d)** を選びます
3. 保存先を指定して **エクスポート** をクリックします

**`.stl` ファイル（3Dプリンタ用）：**

1. **ファイル → エクスポート** を選択します
2. ファイル形式で **STL (.stl)** を選びます
3. **リファインメント** は「高」を推奨します
4. 保存先を指定して **エクスポート** をクリックします

::: tip .f3d と .stl の違い
- **`.f3d`**：Fusion の編集データがすべて残る。後から修正が可能
- **`.stl`**：表面の三角形メッシュのみ。3Dプリンタやスライサーソフトで使用
:::

## Part 2：ペン立ての作成

ここからは、学んだ操作を組み合わせてペン立てを設計します。

### 2-1. サンプルファイルのインポート

1. クラスルームで配布されたペン立てのサンプル `.f3d` ファイルをダウンロードします
2. Fusion で **ファイル → 開く** またはデータパネルから **アップロード** します
3. サンプルを参考に、各部の寸法や構造を確認します

### 2-2. 直方体の作成

1. 新しいプロジェクトを作成します
2. **ソリッド → 作成 → 直方体** で以下のサイズの直方体を作成します：
   - 幅：**75 mm**
   - 奥行き：**70 mm**
   - 高さ：**100 mm**

### 2-3. 空洞部分の切り取り

ペン立ての内側を空洞にします。

1. 直方体の **上面** をクリックしてスケッチを開始します
2. **スケッチ → 修正 → オフセット** を使い、上面の外周から内側に **2 mm** オフセットした長方形を描きます
3. **スケッチを終了** します
4. オフセットした長方形を選択し、**押し出し** で切り取ります
   - 方向：下方向
   - 距離：**-98 mm**（底の厚みが 2 mm になるように）
   - 操作：**切り取り**

::: tip 壁の厚みについて
2 mm はこの程度のサイズの造形物では十分な強度があります。厚くすると強度は上がりますが、造形時間と材料が増えます。
:::

### 2-4. 前面を斜めにカット

ペンが取り出しやすいように、前面を斜めにカットします。

1. 直方体の **右側面** をクリックしてスケッチを開始します
2. **線分** を使って、左上の角から以下の直角三角形を描きます：
   - 下方向に **45 mm**
   - 右方向に **35 mm**
3. **スケッチを終了** します
4. 三角形のプロファイルを選択し、**押し出し** で **切り取り** します
   - 方向：全体を貫通するように（「すべて」を選択）

### 2-5. 文字の彫刻

ペン立ての前面に名前やイニシャルを彫り込みます。

1. ペン立ての **前面（斜めにカットした面）** をクリックしてスケッチを開始します
2. **スケッチ → 作成 → テキスト** を選択します
3. 任意の文字を入力し、フォントやサイズを調整します
4. **スケッチを終了** します
5. テキストのプロファイルを選択し、**押し出し** で **切り取り** します
   - 深さ：**-1 mm**

::: warning 3Dプリントの注意点
宙に浮いた部分（オーバーハング）はサポート材が必要になります。文字のデザインは、なるべくオーバーハングが少ないものを選びましょう。どうしても避けられない場合は、スライサーソフトでサポートを追加するか、パーツを分割して造形後に組み合わせます。
:::

### 2-6. フィレット

角を丸めて仕上げます。

1. **ソリッド → 修正 → フィレット** を選択します
2. 丸めたい **エッジ（辺）** を選択します（側面や上面の角を選びましょう）
3. フィレットの半径を設定します（例：**3 mm**）
4. **OK** で確定します

::: tip
フィレットは見た目を良くするだけでなく、3Dプリント時の強度向上にも効果的です。
:::

### 2-7. エクスポートと提出

1. **ファイル → エクスポート** で `.f3d` ファイルを書き出します
2. 同様に `.stl` ファイルも書き出します
3. 両方のファイルをクラスルームに提出します

## 発展課題（任意）

### オリジナルペン立ての作成

基本のペン立てが完成したら、オリジナルのデザインに挑戦してみましょう。

- 形状の工夫（円筒形、多角形、有機的な曲面など）
- 仕切りや小物入れの追加
- 模様や装飾の追加（円形状パターン、格子状パターンの活用）
- 複数のボディを結合した複雑な形状

完成したら `.f3d` と `.stl` をクラスルームに提出してください。

### オリジナルペン立ての3Dプリント出力

オリジナルペン立てを実際に3Dプリンタで出力します。

1. `.stl` ファイルをスライサーソフト（Cura など）で開きます
2. プリント設定を確認します（インフィル密度、サポート材の有無など）
3. 出力が完了したら、写真を撮影してクラスルームに提出してください
