## 今回の作業

### 1. 作品の構想を具体化する

1. どの機構（歯車・カム・揺動クランク等）を使うか決めます
2. 完成イメージをスケッチ（手描きでOK）します
3. 必要なパーツのリストを作成します
4. 各パーツの大まかな寸法を決めます

::: tip 構想のポイント
- まずは **シンプルな構成** で考え、動くことを最優先にしましょう
- 複雑にしすぎると組み立てが困難になります
- 機構の動きを何か別のものに **見立てる** と独創性が高まります（例：歯車の回転 → 惑星の公転）
:::

### 2. パーツの設計

1. Fusion で各パーツをモデリングします
2. パーツ同士の **はめ合い** や **接合部** の寸法に注意します
3. 3Dプリンタ出力の場合：
   - オーバーハングが 45° 以上にならないよう工夫する
   - サポート材が必要な箇所を最小限にする
4. レーザーカッター出力の場合：
   - 板厚に合わせた寸法設計
   - DXF ファイルの書き出し

### 3. パーツの出力

1. 設計したパーツを `.stl`（3Dプリンタ）または `.dxf`（レーザーカッター）で書き出します
2. 出力機器の操作手順に従って出力します
3. 出力したパーツの寸法を確認し、必要に応じて設計を修正します
