# Lesson 10: Gear Mechanism

## Goals

- Design gears in Fusion
- Create a crank arm (handle) and combine it with gears
- Learn to combine gears at right angles (bevel gears)
- Export DXF files for laser cutting

---

## 1. Creating Spur Gears

Use **Solid → Create → Spur Gear** (or search "Spur Gear" in the toolbar) with parameters: Module, Teeth count, Thickness, Pressure angle.

## 2. Crank Arm (Handle)

Sketch and extrude an arm shape on the gear's side face to create a handle for manual rotation.

## 3. Right-Angle Gear Assembly (Bevel Gears)

Use bevel gears to change the rotation axis by 90 degrees. Useful for converting horizontal rotation to vertical.

## 4. Dimension & Position Adjustment

Use **User Parameters** for centralized dimension management and the **Align** command for precise positioning.

## 5. DXF Export

Project body edges onto a sketch plane, then right-click the sketch → **Save as DXF** for laser cutter output.
