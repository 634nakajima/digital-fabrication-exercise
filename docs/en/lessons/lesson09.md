# Lesson 9: Joints & Assembly

## Goals

- Understand the difference between Bodies and Components
- Use Joint features to assemble parts (Rigid, Revolute, Slider)
- Understand bottom-up vs. top-down design approaches

---

## 1. Components

- **Bodies** are shapes within a single part; **Components** are independent parts that can be jointed
- Right-click a body → **Create Component** to convert it
- Two copy types: **Copy/Paste** (independent) and **Linked Copy** (synchronized)

## 2. Rigid Joints

1. Select **Assemble → Joint**
2. Click snap points on both components
3. Set motion type to **Rigid**
4. Practice: Assemble a box from flat panels

## 3. Revolute & Slider Joints

- **Revolute**: Allows rotation around an axis
- **Slider**: Allows linear sliding movement
- Set motion limits via **Edit Joint → Motion** tab
- Practice: Assemble a box with rotating bar, sliding bar, and stopper
